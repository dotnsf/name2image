# name2image

## Overview

**『名前』を入力すると、その『名前からイメージされる画像』を AI が検索して結果を表示する** というウェブアプリケーションです。


## Pre-requites

IBM Cloud のアカウントを取得して、 Watson Language Translator サービスインスタンスを作る必要があります。また、作成したサービスインスタンスに対して、以下のいずれかの手順を行っておく必要があります：

1. （IBM Cloud 上のランタイムとしてアプリケーション・サーバーを稼働させる場合のみ）該当ランタイムから Watson Language Translator サービスをバインド

2. Watson Language Translator サービスの環境変数を参照し、apikey 及び url の値を settings.js 内の該当部分にあらかじめ転記
